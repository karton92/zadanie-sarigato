<h2>Installation</h2>

_Below is how install and setting up app in terminal._

#### 1. Clone the repo
   ```sh
   git clone https://github.com/karton92/zadanie-sarigato.git
   ```
#### 2. Install NPM packages
   ```sh
   npm install
   ```
#### 3. Run application
   ```sh
   1. npm run server
   2. npm start
   ```
